title: 模块 - 介绍
---

本文档分为四个部分：

1. 第一部分阐述 TypeScript 处理模块的 [**理论基础**](/modules-reference/theory)。如果你想能够为任何情况编写正确的模块相关编译器选项，推理如何将 TypeScript 与其他工具集成，或理解 TypeScript 如何处理依赖包，这是开始的地方。虽然这些主题有指南和参考页面，但建立对这些基础知识的理解将使阅读指南变得更容易，并为你提供一个处理未在此具体涵盖的真实问题的思维框架。
2. [**指南**](/modules-reference/guides/choosing-compiler-options) 展示如何完成特定的实际任务，从为新项目选择正确的编译设置开始。对于希望尽快上手的初学者，以及已经很好地掌握理论但希望在复杂任务上获得具体指导的专家来说，指南是一个很好的起点。
3. [**参考**](/modules-reference/reference) 部分提供了对前面部分介绍的语法和配置的更详细说明。
4. [**附录**](/modules-reference/appendices/esm-cjs-interop) 涵盖复杂主题，这些主题需要比理论或参考部分更详细的解释。
